# Emotion Analysis using NLP and Fuzzy Logic

## Description
This project implements a hybrid emotion analysis system that combines natural language processing (NLP) techniques with fuzzy logic to classify emotional content in text. The system generates text from prompts and then evaluates it using a secondary model to determine its emotional state.

## Key Features
- Text generation and evaluation pipeline
- Emotion classification with 5 levels:
  - Very Sad
  - Sad
  - Neutral
  - Happy
  - Very Happy
- Word embeddings using Word2Vec for semantic representation
- Fuzzy logic implementation with membership functions
- Discretization of emotional states for improved interpretability

## Methodology
1. Generate text from input prompts
2. Convert text into vector representations using Word2Vec
3. Apply classification model to detect emotional content
4. Use fuzzy logic and membership functions to map outputs into graded emotional levels

## Technologies
- Python
- NLP techniques
- Word2Vec
- Fuzzy Logic
- NumPy / Scikit-learn

## Applications
- Sentiment analysis
- Human-computer interaction
- Emotion-aware AI systems
